Checking headings
 

Cindex can compare headings in successive records, and identify records in which the differences between records are probably errors. Checks on consistency of headings are available as part of a comprehensive check via Check Index… (Tools menu).

Open the panel, click the Headings tab, and configure the checks you want:

  • Inconsistent capitalization. Finds headings that are identical except for letter case.
  • Inconsistent style/typeface. Finds headings that are identical except for text style (bold, italic, etc.) and/or font.
  • Inconsistent punctuation. Finds headings that are identical except for punctuation.
  • Inconsistent leading conjunctions/prepositions. Finds subheadings headings that differ in a leading conjunction or preposition. When making this check, Cindex uses conjunctions and prepositions from the list for making smart flips in records (Document menu, Smart Flip Words…)
  • Inconsistent Endings:
    • Plurals. Finds headings that have inconsistent plural endings.
    • Conjunctions/prepositions. Finds headings that differ in a terminal conjunction or preposition. When making this check, Cindex uses conjunctions and prepositions from the list for making smart flips in records (Document menu, Smart Flip Words…)
    • Parenthetical phrases. Finds headings that are identical except for any ending phrase in parentheses.
  • Orphaned subheadings. Finds records in which a subheading below the specified level is redundant (i.e., is unique, so the content could be pulled up to the level above).
  • Modifying phrases that could be subheadings. Finds records in which a modifying phrase in a heading might more appropriately be a subheading.

If Cindex finds errors it displays them in the check results. Each entry in the list identifies the source record and the nature of the error.

Click any item in the list to bring its record into view in the main document window; double-click the item (or hit RETURN) to open a window in which to edit its record.
