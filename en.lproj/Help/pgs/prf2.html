Settings for viewing entries
 

Use the Preferences… panel (Cindex menu) to set preferences.

Settings you make under the View tab control how Cindex displays entries in the main window and in the record-editing window.

•

Default Font establishes the font Cindex will use to display entries in the record window and the main window (unless you are displaying the index in formatted view, and have explicitly specified other fonts for headings — See Document->Headings).

•

Main View establishes the size of type Cindex will use to display entries in the main window (unless you are displaying the index in formatted view, and have explicitly specified other sizes for headings — See Document->Headings). If you specify 0, Cindex will use 12 point type.

•

Entry View establishes the size of type Cindex will use to display entries for editing in the record window. If this is 0, Cindex uses the size you have set for the main window (above).

•

Label colors determines the colors Cindex will use to display records that carry different labels. Click a color well to set the color for the corresponding label.

To make Cindex display labels in a fully formatted view, check Show in Formatted View.
