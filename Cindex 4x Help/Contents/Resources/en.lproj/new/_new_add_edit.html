New features: adding and editing records
 

Demoting Headings
You will sometimes want to gather entries that have different headings, but are related, under a common main heading. Cindex now provides a command, Demote Headings, that enables you to do this in a single step, without your needing to edit the records individually.

Splitting Headings
When you make an author index, you will often import several names in a single operation, e.g., by copying and pasting names from a list of references, or importing a list of references that contains names, possible with additional text such as dates. The record heading that results then has to be edited to remove superfluous text, often requiring the generation of additional records, each containing a single name. Cindex can now do all this automatically with the Split Headings command.

Pasting and Dropping Text
In earlier versions of Cindex, pasting or dropping text into a record carried the style and font attributes of the text, unless you held the SHIFT key down while completing the operation, in which case style and font attributes were discarded. Cindex now provides finer control of this. You can control whether Cindex pastes both style and font information, styles only, or plain text. Moreover, you can set a Preference for the default behavior.

Adjusting Page References
When you use the Alter command to adjust page references, and use a pattern to specify that leading text in a reference should be skipped, Cindex now applies the pattern to both parts of a page reference range.

Managing Fonts
A new item in the contextual menu available when you are adding or editing records provides quick access to fonts you have already used in the index.
