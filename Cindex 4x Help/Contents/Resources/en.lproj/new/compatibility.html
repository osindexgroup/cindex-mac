Compatibility
 

Cindex 4 is fully compatible with Cindex 3, and uses the same document format. If you are upgrading from Cindex version 2 or earlier, Cindex 4 will convert documents as needed when it opens them.

When Cindex 4 opens an index made by version 2 or earlier, it will make a copy of the index and convert the copy to the new format. The old index document will not be altered.

An index in Cindex 4.0 format cannot be opened by Cindex 2 or earlier. If you need to provide someone with an index that will work with Cindex 2 or earlier, save it as an archive.

Navigation and keystrokes
Some of the special characters used in pattern-matching are changed from Cindex version 2.
