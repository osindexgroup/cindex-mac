Formatting cross-references
 

Use the Cross-References panel (Document menu) to specify the formatting and punctuation of references.

Cindex can place cross-references in different positions, depending on your needs.

To specify how cross-references should be placed and punctuated, make settings in the From Main Heading tab or the From Subheading tab.

To specify type styles to be applied automatically to the introductory phrase of a cross reference, click Prefix Style.

To specify type styles to be applied automatically to the body of a cross reference, click Body Style. If you choose Title Case for capitalization, Cindex capitalizes words in headings as follows:

  • The initial letter of every word is capitalized, unless preceded by ~.
  • Every letter after the first in a word is set in lowercase, unless preceded by ~.
  • Words in the list for smart flipping (Smart Flip Words… from the Document menu) are not capitalized.

To ensure that multiple cross-references from an entry are arranged in sorted (normally ascending) order, check Arrange Alphabetically.

To suppress the display of all cross-references in the formatted index, check Suppress All References.

To prevent Cindex applying an automatic style to the cross-reference prefix if that style is already applied to the body of the cross-reference, check Don't Apply Prefix Style if it Matches Existing Body Style.

If you set the cross-reference format when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
