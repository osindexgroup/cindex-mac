Creating an index
 

To start a new index choose New… from the File menu. Cindex will display a panel that asks you to name the index. (Unlike a word-processor, Cindex does not work with a new “untitled” document—you must create and name a new document before you make entries). It is often useful to give the index a name that identifies the material being indexed. Click Save to create the index.

When you create an index, Cindex provides standard settings for certain properties of the index records that will contain entries. For example, Cindex sets a limit on the number of characters that a record can hold. You can change these properties before you create the new index, or at any time afterward. To change the properties of the records before you create the new index, click Options... before you click Save.

When you click Save, Cindex will open a new document window in which the index entries are displayed as you make them. Most of your commerce with the index will be through this window. When you close this window, you close and save the index document.

Cindex can also create a new index from a template—an existing index that serves as a model for the new one. The new index inherits all its settings from the template.
