Heading typography and punctuation
 

Use the Headings panel (Document menu) to specify the typography and punctuation of headings.

When displaying or printing a fully-formatted index, Cindex can show headings and subheadings in any font and style you choose, and can provide custom punctuation for them. Cindex can also suppress the display of headings at specified levels. You can provide different settings for each level of heading. Choose from the popup menu the level for which you want to make settings. If you have specified (by setting Record Structure) that index records will contain a required last subheading, the format settings you make for the last level on the menu will be used to format the special subheading, regardless of its actual depth in a record.

From the Font pop-up menu choose the font. The first item in the menu (<Default>) stands for the standard font used in the main view. Cindex is initially set to use this font for all levels of heading.

From the list to the right of the Font pop-up menu choose (or type in the box) the size in points of the type in which the heading will be displayed. Choose 0 to use the standard size set for the body of the index.

Click Style… to specify the style in which you want the heading displayed. If you choose Title Case for capitalization, Cindex capitalizes words in headings as follows:

  • The initial letter of every word is capitalized, unless preceded by ~.
  • Every letter after the first in a word is set in lowercase, unless preceded by ~.
  • Words in the list for smart flipping (Smart Flip Words… from the Document menu) are not capitalized.

To make Cindex show leading text immediately before the heading provide that text at Leading Text.

To specify closing punctuation or other characters that Cindex will append to a heading that has no immediately following page references or cross-references, enter the text at Trailing Punctuation.

To suppress the display of the heading, check Suppress. Cindex will suppress the heading and any associated page references and cross-references.

If you set heading typography and punctuation when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
