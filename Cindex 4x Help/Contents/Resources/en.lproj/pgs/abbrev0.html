Using the Abbreviations panel
 

Use the Abbreviations panel (Tools menu) to manage abbreviations that you use in records.

The table shows all current abbreviations and the expanded phrases they represent.

To add an abbreviation you must provide both its name and the phrase it represents. The name of the abbreviation may contain any sequence of up to eight characters except a space and these punctuation marks:

( ) [ ] " “ ” ' ‘ ’ , . ; :

Click the + button to define an abbreviation. Cindex will open a cell in the table, into which you should type. Click the cell to the right to enter the phrase the abbreviation will represent. You can apply styles to the text of the phrase.

You can also define abbreviations directly in records as you work on them.

To edit the name of an existing abbreviation, or its expanded phrase, double-click in the table.

To remove an abbreviation, click to select its row in the table and clikc the – button or choose Delete from the Edit menu.

You can view abbreviations sorted alphabetically by abbreviation or by expanded phrase. To view table entries in order of abbreviation, click the title of the Name column (click the title again to switch between ascending and descending sort). To view the table entries in order of expanded phrase, click the title of the Expansion column.

To close the active set of abbreviations, click Close Set.

To begin a new set of abbreviations, click New Set… Cindex will close the current set and establish a new empty set. The new set becomes the active set of abbreviations.

To load an existing set of abbreviations (and make it the active set) click Load Set….

Cindex keeps track of the active set of abbreviations, and automatically uses it when you start a new session.
