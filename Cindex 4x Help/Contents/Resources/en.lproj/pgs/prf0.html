General preferences
 

Use the Preferences… panel (Cindex menu) to set preferences.

Settings you make under the General tab control the overall behavior of Cindex.

•

To specify what Cindex will do when it starts up, click a button under Startup.

Check Prompt for User ID if you want to be prompted for your identifying initials, which will be attached to each index record you add or change. (If you don't often change your identifier, you should leave this unchecked, and set the identifier as described below.)

•

To specify the interval at which Cindex will automatically save an open index to disk, set the number of minutes in Auto Save Index Every N minutes. If you set the interval to be 0, Cindex will never save the index automatically.

•

To provide a user identifier that Cindex will attach to each record you make or change, enter up to four characters at User ID.

•

To specify the measurement unit Cindex normally uses for displaying dimensions, choose from Measurement Unit.

•

To set Cindex so that it will notify you automatically when an update is available, check Automatically check for updates. Cindex requires an active internet connection when it checks for updates. You can also check manually at any time by choosing Check for Update from the Cindex menu.
