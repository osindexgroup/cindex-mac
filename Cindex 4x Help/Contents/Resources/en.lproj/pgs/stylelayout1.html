Setting indentation of entries
 

Use the Style and Layout panel (Document menu) to specify the index style, and the layout of entries.

Settings for indentation are in the Indentation box. Choose from the pop-up menu the method for indenting entries:

None

Cindex provides no leading or runover indentation.

Incremental

Cindex indents the start of each subheading by a specified amount that depends upon the subheading level; for any runover lines Cindex adds an additional amount to the lead indent. You specify the runover indentation and the lead for the first level of subheading, and Cindex calculates all other indents automatically.

Fixed

Cindex indents headings and runover lines by amounts that you specify separately for each level of heading. This method provides the greatest control over indentation, but requires more settings than others. Choose Fixed to use this method. If you choose fixed indentation Cindex displays a popup menu through which you choose the heading(s) for which you will set indentation.

Special Last

This method is like standard incremental indentation, but the lowest level of subheading in an entry (regardless of its actual level) has fixed lead and runover indents. This method is designed for indexes in which the body of an entry must be indented by a fixed amount regardless of its placement under a main heading or a subheading

When using Fixed indentation, you should specify a lead and runover indent for each level of heading that will appear in your index. Choose the heading from the Level popup menu, then set the lead and runover indents. Cindex indents each heading and runover line by exactly the Lead and Runover amounts you have specified for each level.

When using Special Last indentation, you must specify incremental indents that will be used for all levels of heading except the last. You must also specify fixed indents that will be used for the lowest-level headings.

Specify the lead and runover indents in the Lead and Runover boxes. You can use fractional values for both the lead and runover indents. Choose from the Unit pop-up menu the unit in which you want to specify indents.

When using Incremental indentation, you need specify only one lead indent and one runover indent. Cindex indents each heading by its level x Lead indent; for runover lines Cindex adds the Runover indent to the lead indent.
