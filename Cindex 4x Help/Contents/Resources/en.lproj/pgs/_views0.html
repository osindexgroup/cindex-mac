Viewing the index
 

Cindex provides several ways for you to view the index as you work on it. These range from a fully formatted view that lays out the index exactly as you would see it printed in a book to an unformatted view that makes explicit the structure of the records from which the index is formed. Cindex also allows you to display and work with only parts of the index.

If you are working in a right-to-left reading language such as Arabic or Hebrew, Cindex will lay out entries correctly as long as it knows the language you are using. Specify the language via Sort settings in the Tools menu. You can override the natural layout by choosing from the Text Alignment submenu in the Format menu.

The title of a main document window is always the name of the index. The toolbar underneath the title contains tools that control how the index is displayed. You can add or remove tools (and specify how they are displayed) by choosing Customize Toolbar… from the View menu. To learn the function of a tool, move the mouse pointer over it.

To change the font in which the index is displayed, choose from the font list in the toolbar.

To change the type size in which the index is displayed, choose (or type) in the type size list.

The status bar at the bottom of the window displays:

•

Which records are accessible in the view.

•

How many records are accessible in the view.

•

How many new records have been added in your current session.

•

Whether the entries are sorted.

Use items in the View menu to choose the kind of view you want.
