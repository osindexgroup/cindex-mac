Labeling records
 

Cindex lets you label records that it will display in a conspicuous color. A label serves as a marker you can use for any purpose you choose (for example to identify records that contain incomplete entries). A labeled record is displayed in a distinctive color of your choosing.

You can use up to 7 different labels to tag records (an individual record may have only one label).

To label a record (or records), select them in the main document window then choose from the Labeled submenu of the Edit menu, or choose from the menu attached to the toolbar button (or right-click and choose from the contextual menu).

To remove a label from a record or records, select the record(s) then either:

  • Choose from the menu the label you want to remove. This will remove only the chosen label from records that bear it. It will not affect any selected records that carry different labels.
  • Choose No Label from the menu. This will remove all labels from selected records.

If the record-entry window is open, you can label or unlabel the current record by choosing from the Labeled submenu. If the current record is labeled, the status line at the bottom of the record window displays “Labeled N.”

•

When you add or change a label, Cindex by default does not mark the record as 'modified' (i.e., it does not update the date stamp and user identifier). To specify that a label change is considered a modification, choose Preferences from the Cindex menu, click the Edit tab, then check Label Sets Change Flag.

•

To change the color Cindex uses for a particular label, choose Preferences from the Cindex menu and click the View tab, then click the color well for the relevant label. Cindex will display a panel through which you can choose the color you want.

•

Cindex normally applies the label color to records only when displaying index entries in a draft view. To make Cindex use the label color in the fully formatted view, choose Preferences from the Cindex menu, then on the View tab check Show in Formatted View.
