Deleting and restoring records
 

To remove a record (or records) from the index, select them in the main document window then check Deleted in the Edit menu. A deleted record is not physically removed from the index—it merely becomes hidden when you view a fully formatted index. In a draft or unformatted view, a deleted record is shown with the symbol ≠ (in blue) in the left margin beside it. Deleted records are never displayed in a formatted view.

You can restore deleted records by selecting them in the main document window and unchecking Deleted in the Edit menu.

If you are working in the record window you can delete the current record by checking Deleted in the Edit menu. You can restore a deleted record by unchecking Deleted. The status line at the bottom of the record window displays “Deleted” if the current record is deleted.

You can remove deleted records permanently by choosing Compress… from the Tools menu. Deleted records cannot be restored after you have compressed the index.
