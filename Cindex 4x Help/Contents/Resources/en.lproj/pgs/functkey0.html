Using the Function Keys panel
 

Use the Function Keys panel (Tools menu) to manage frequently-used words or phrases that you can insert in records when you hit function keys (F1 through F16).

You can attach frequently-used words or phrases to up to 16 keys, and by hitting the appropriate key you can insert the text in a record. The text you attach to a key can contain multiple style attributes such as boldface and italics.

To review or edit the text attached to a key, double-click the text in the table. You can change the text attributes (boldface, italics, etc.) but you cannot change the font or type size.
