Options for saving formatted documents
 

To set options for saving formatted, click Options in the panel Cindex presents when you choose Save To from the File menu.

Settings you make here control how Cindex saves formatted entries in a document.

•

Items under Scope define the collection of entries to be saved. For more information on this see Specifying the scope of a command.

•

Application programs that read and write text documents use a particular character to denote the end of a paragraph (in Cindex, a formatted index entry). The character traditionally used by Macintosh programs is not the same as that used by programs that work under Windows. Some document-management systems can deal with text documents that use either convention, but others will accept documents of only one type—Macintosh or Windows.

Click Macintosh to make a document in which the Macintosh convention is used to define the end of a text paragraph; click Windows to use the convention employed by other kinds of computers.

•

When saving documents in Rich Text Format, or in the formats required by QuarkXPress or InDesign, Cindex can define leading indents to subheadings in one of several different ways.

Click the appropriate button beside Subhead Indent. You can choose to have indents defined as an attribute of the style attached to the subheading, by using ‘tab’ characters (one for each level of subheading), or by using a character of your choice (one for each level of subheading).

•

By default, Cindex identifies levels of heading in the exported index using the names you set for record fields in the index. To use different names, click Paragraph Style Names… Cindex will present a panel through which you can set, and enable, the names you want to use for headings in the exported document. The names you provide are specific to the type of document you are creating (e.g., QuarkXPress or InDesign). Cindex remembers the names, even if you are not using using them.

You can specify default settings by choosing Preferences… (Cindex menu) and clicking the Export tab.
