Sorted and unsorted views
 

Cindex can display index entries in sorted order or in the order in which you entered them in the index.

To view records in sorted order, check Sorted in the View menu. Cindex provides many ways for you to sort index entries. Cindex is set initially to display sorted entries in alphabetical order. You can quickly switch between an alphabetical sort and a page order sort by clicking the relevant button in the main toolbar.

To display records in record number order, uncheck Sorted in the View menu. To display entries in whatever order is dictated by the sort rules in effect, check Sorted in the View menu.

When you view records in a group, Cindex always display them in the order specified for the group, and the Sorted menu item is dimmed. (You can change the order by sorting the records in a group.)
