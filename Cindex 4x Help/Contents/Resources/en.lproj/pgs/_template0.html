Making and using index templates
 

When Cindex creates a new index it can use the record structure and other standard settings (for sorting and formatting entries) from an existing index. To use one index as a model for another, you must first create a template. You can make a template in two ways:

•

In the Finder, select the index you want to use as a template, then choose Get Info from the File menu. The Finder will open a window that displays information about the document. To make stationery from it, check Stationery Pad. Close the window when you are done.

•

When Cindex is active, you can make a template that is based on the active index. Choose Save To… from the File menu, then select Template from the pop-up menu. When you save an index as a template, Cindex saves information about the structure of records, and the other settings that belong to the active index, but saves no records.

When you open a template with Cindex (through Open… from the File menu), Cindex will ask you to provide a name for a new index. When you provide the name Cindex will create the new index with the settings contained in the template.

When you open a template through the Finder, the Finder will create a copy of the file and then open it through Cindex.
