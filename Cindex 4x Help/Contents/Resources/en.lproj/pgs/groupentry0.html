Grouping entries
 

When it displays or prints a fully-formatted index, Cindex can set-off different alphabetical groups of entries by printing a title above the group (typically the letter of the alphabet under which entries in the group fall). Use the Grouping Entries panel to specify how alphabetical groups of entries should be laid out.

At Text, enter any text you want to appear at the head of each group of entries (if you supply no text, Cindex displays no title). Use the character %, on its own or as part of a longer text sequence, to stand for the leading letter of the group of headings. When Cindex displays the title it will replace the % with the appropriate letter or symbol.

Cindex always treats each letter of the alphabet as belonging to a distinct group, but lets you choose how it groups entries beginning with numbers and symbols. Choose from the Grouping Method pop-up menu the method you want it to use:

[@] [#] [$]… [1] [2] [3]…

each leading symbol or numeral forms a distinct group

[@#$…] [1] [2] [3]…

all leading symbols form one group; each numeral forms a distinct group

[@] [#] [$]… [123…]

each leading symbol forms a distinct group; all numerals form one group

[@#$… 123…]

all leading symbols and numerals form a single group

[@#$…] [123…]

all leading symbols form one group; all numerals form one group

Enter at All Numbers Title, etc., the text to be displayed at the head of entries that are grouped generically under numbers, or symbols, or numbers and symbols.

To specify the font in which the title will be displayed or printed, choose from the Font pop-up menu. Choose <Default> to use the standard font in which entries are displayed. To specify the type size in points, choose from the list to the right of the Font pop-up menu (or type in the box). Choose 0 to use the standard size set for the body of the index.

To specify a style in which the text should be displayed, click Style… Cindex will display a panel through which you can specify the style in which the title is to be displayed.

If you make settings for grouping entries when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
