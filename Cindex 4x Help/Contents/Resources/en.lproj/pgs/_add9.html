Using function keys
 

If your keyboard has a top row of keys labeled F1 etc., you can attach text to these so that with a single keystroke you can enter the text in records.

You can attach text to a function key whenever a record is open for editing. To do this, select the text you want to attach (this can span more than one field), then, while holding down the COMMAND and OPTION keys, hit the function key to which you want to attach the text. For example, to attach the selected text to the F5 key, use COMMAND OPTION F5. Use the same procedure to change any text already attached to a key. You can attach text to any of the keys F1 through F16. The function key attachments apply to all indexes, and Cindex remembers them until you change them.

To insert text in a record, click where you want the text to appear (or select any text you want to replace), then hold the OPTION key down and hit the appropriate function key. If the text you want to insert would cause the record to contain too many fields, or too many characters, Cindex will warn you with a beep.

To review or edit the text attached to the function keys, choose Function Keys… from the Tools menu. Cindex will open a panel in which it displays, beside the name of each key, the text attached to it. You can leave this panel open while you work on the index.
