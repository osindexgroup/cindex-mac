Altering references with patterns
 

When you use Alter References (Tools menu) to adjust page references, Cindex normally changes the first number (or range) that falls within the range you specify. If records contain multi-part page references you will probably want to restrict the alterations to a particular segment. You do this by using a pattern to direct Cindex to skip to the segment you want to alter.

In the Scope box, specify the pattern in the Matching Pattern box. This pattern should describe the initial segment of each reference that should be skipped before an adjustment is made, as in the following examples:

•

To deal only with with references to volumes for 1988 in the following compound references:

1987: Jun 555; 1988: Oct 1032; 1989: Jan 22

define a pattern that identifies the 1988 volume:

1988:[^1-9]*

•

To alter only the page number (the last element) in the following compound references:

1988(3)55, 1988(4)66, 1988(5)72, 1988(4)60-63

use this pattern to exclude all components of the reference up to the closing parenthesis:

.*\)
