Special characters
 

The characters < > { } ~ and \ have special significance in the sorting of entries. To include one of these reserved characters as a literal character in a record, place \ immediately before it. Thus, to use \ as a literal character you would type \\ in a record; to use ~ as a literal character you would type \~. When Cindex displays the index entry it shows only a single \ or ~.

A record may contain any character that you can type on the keyboard (including characters that are formed when you hit a key while holding down the option key). You can also choose characters from the character palette. To display the palette choose Emoji & Symbols… from the Edit menu.

To insert a character from the palette into a record, click in the record window at the place where you want to insert the character, then double-click the character in the palette.
