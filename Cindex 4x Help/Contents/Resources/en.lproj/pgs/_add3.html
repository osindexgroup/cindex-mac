Saving or discarding changes to a record
 

When you have completed work on a record, hit COMMAND RETURN (or ENTER on the numeric keypad). Cindex will close the window and display the new record in the main document window, formatted in the manner dictated by current settings. If you have tried to enter a new empty record Cindex will discard it automatically.

You can also save an entry by clicking the close button on the record window, or by choosing Close from the File menu. If you use either of these methods, Cindex normally asks if you want to save (or discard) the record. You can set Cindex so that it will not query you, but instead always discards changes to the record, or always saves changes to the record.

To save a new or changed record, and leave a copy of it as the foundation for another record, hit OPTION PAGE DOWN.

To discard changes to a record and restore it to its original state, hit OPTION ESC.

To discard changes to a record and close the record window, hit ESC.
