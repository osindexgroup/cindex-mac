Formatting page references
 

Use the Page References panel (Document menu) to specify the formatting and punctuation of page references.

When Cindex displays a fully-formatted index, it sorts, formats and punctuates page references. Cindex first gathers all the references from identical records. It then discards any duplicates, and any redundant references to single pages that match either the beginning or the end of an inclusive range of pages. After doing this it can conflate a series of separate references to consecutive pages, it can abbreviate page ranges, and it can suppress repeated volume numbers in multi-part references.

To specify how ranges page ranges are formatted, make settings in the Ranges box.

To specify how multi-part locators are formatted, make settings in the Multi-Part References box.

To specify how page references are punctuated, make settings in the Punctuation box.

To specify type styles to be applied automatically to page references, click Style.

To ensure that multiple page references from an entry are arranged in sorted (normally ascending) order, check Arrange in Sorted Order.

To hide duplicate or redundant references in the formatted display, check Merge Redundant References. This option is available only when references are displayed in sorted order.

To suppress the display of all page references in the formatted index, check Suppress All References.

If you set page reference format when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
