Alphabetizing text as some other text
 

If you know that you will always want to spell out certain words (e.g., St. as Saint), or will always want to ignore certain words or phrases, you can set Cindex to do this automatically, without needing to insert override text in {…} or <…>. You manage this by specifying a set of text pairs: target text that Cindex looks for in records, and substitute text used in its stead when alphabetizing.

To manage substitutions click Substitutions… on the Text tab of the Sort panel. A new panel presents a list of substitution pairs.

Double-click an item in the left side of the list to enter or edit the target text (e.g., St.) that should be alphabetized as something else (e.g, Saint). To the right enter or edit the substitute. If the target text is to be ignored entirely, provide no substitute text. To add a row, click +. To delete a row, select it, then click –.

There are some restrictions on text you can enter in the list:

  • Target text (on the left) must be a complete word or the start of a word (i.e., it must begin with a letter or a number). It may be a prefix to its own substitute (on the right), but it may not match, or be a prefix to, any other text in the list.
  • The substitute text (on the right) may not begin with text that matches any target except its own. It may be empty, which causes Cindex to ignore the target text when alphabetizing entries.

If you provide an invalid target or substitute, Cindex will beep, and will require you to correct the error before it completes the edit.

When Cindex alphabetizes records, it will find a target only when the target text begins a word, or matches a whole word. Cindex will not find a match if the target text sits within {…} or <…>.

For each target found, Cindex treats its text as if it were the substitute. For example, with 'Saint' set as the substitute for 'St.' this record field:

St. George’s Cross

would be alphabetized as though its text were

Saint George’s Cross

Cindex attends to substitutions only in letter-by-letter or word-by-word alphabetizing. Substitutions are ignored in a simple sort.
