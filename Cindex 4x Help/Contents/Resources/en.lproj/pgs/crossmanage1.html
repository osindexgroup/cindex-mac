Generating cross-references
 

Cindex can generate cross-references automatically, so that the existence of a specified entry can automatically lead to the appearance of a cross-reference.

To generate automatically a set of cross-references to specified headings in your index, you must build a special reference index that identifies the target entries and the cross-references that should be made to them.

Each record in the special cross-reference index should be organized as follows:

•

In the main heading field, enter the text that will be the main heading of the cross-reference.

•

In the first subheading field enter any subheading that accompanies the main heading. You can continue this to whatever depth of sub-subheading you wish.

•

In the locator field enter the text of the target entry, without any leading See or See also. Cindex will decide what form of introductory phrase is required when it includes the cross-reference in the destination index. If you want to refer to a subheading under a main heading, enter the main heading with the text of the subheading as a modifier, after a comma.

To use the special index to generate cross-references for the active index, choose Manage Cross-References... from the Tools menu, then choose Generate from File… Cindex will display a panel through which you open the index that is the source of cross-references that Cindex will automatically place in your current index whenever it encounters an appropriate target entry.

Cindex normally generates both See and See also references, as appropriate, but if you want to create only See references (omitting any See also references) check “See” References Only before opening the index.

Cindex knows which records contains automatically generated cross-references. To find these, choose Find, and check Generated Records. (If you edit and change a generated record, Cindex no longer considers it to have been generated automatically, and you will not be able to identify it with Find.) You can count generated records by choosing Count… from the Tools menu; you can remove them by choosing Compress… from the Tools menu.
